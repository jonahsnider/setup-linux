# setup-linux

Sets up my Linux.

Should be run on a recent version of Ubuntu.

```sh
# You must be in the setup-linux directory for things to work
sudo bash ./init.bash
```

## What it does

Installs a bunch of tools and programming languages.
Configures other things too.

- Nim (via choosenim)
- Node.js
  - Yarn
- Rust
  - Starship via cargo
- SDKMAN!
  - Java
  - Gradle
- Fish
  - Fisher
- .gitconfig
